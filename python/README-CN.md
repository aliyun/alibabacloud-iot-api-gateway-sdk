## Alibaba Cloud IotApiGateway SDK for Python


## 安装

- **使用 pip 安装(推荐)**

如未安装 `pip`, 请先至pip官网 [pip user guide](https://pip.pypa.io/en/stable/installing/ "pip User Guide") 安装pip .

```bash
# 安装 IotApiGateway SDK
pip install alibabacloud_iot_api_gateway
```
